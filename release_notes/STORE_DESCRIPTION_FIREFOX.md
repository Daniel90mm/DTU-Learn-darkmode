# DTU After Dark (Firefox Add-ons Description)

DTU After Dark adds useful enhancements to DTU platforms while keeping the interface clean and readable.

What it includes:
- Dark mode for DTU web systems with preserved readability and key colors.
- GPA tools on CampusNet grades pages:
  - weighted GPA row,
  - hypothetical-grade simulator.
- Live bus departure widget (Rejseplanen data) on DTU Learn.
- Course utilities on `kurser.dtu.dk`:
  - historical grade statistics,
  - textbook linker for literature sections,
  - one-click search in DTU Library and Google Books.

Supported domains:
- `learn.inside.dtu.dk`
- `campusnet.dtu.dk`
- `kurser.dtu.dk`
- `studieplan.dtu.dk`
- `evaluering.dtu.dk`
- `karakterer.dtu.dk`
- `sites.dtu.dk`
- `sts.ait.dtu.dk`
- `s.brightspace.com`

Privacy:
- No personal data is collected or transmitted.
- No tracking and no third-party analytics.
- Preferences are stored locally.
