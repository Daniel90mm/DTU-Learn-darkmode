# DTU After Dark (Chrome Web Store Description)

DTU After Dark adds practical quality-of-life features for DTU students and staff across DTU web systems.

Features:
- Two-tier dark mode for DTU sites (`#1a1a1a` and `#2d2d2d` style levels).
- CampusNet GPA tools:
  - weighted GPA summary,
  - projected GPA simulator with hypothetical grades.
- Rejseplanen bus departures directly on DTU Learn.
- Course tools on `kurser.dtu.dk`:
  - grade statistics,
  - textbook linker for course literature,
  - DTU Library and Google Books search badges.

Supported domains:
- `learn.inside.dtu.dk`
- `campusnet.dtu.dk`
- `kurser.dtu.dk`
- `studieplan.dtu.dk`
- `evaluering.dtu.dk`
- `karakterer.dtu.dk`
- `sites.dtu.dk`
- `sts.ait.dtu.dk`
- `s.brightspace.com`

Privacy:
- No personal data collection.
- No analytics or tracking.
- Settings are stored locally in browser storage.
